# 开发任务跟踪

## 任务描述
为 ToDoList Web App 添加时间快捷按钮功能

## 需求
1. 在新建待办事项的时间选择下方添加快捷按钮：
   - [x] 下班前（当天17:30）
   - [x] 明天前（明天17:30）
   - [x] 周五前（如果今天在周五前就本周五，如果今天是周末就下周五）
   - [x] 下周前（下周五17:30）

2. 规范所有时间的时区为北京时间
